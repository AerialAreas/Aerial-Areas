# About
Geometry shaped aliens are invading Earth. You must stop them by finding the area or perimeter of their ships.

# Features
- Enter solutions to area or perimeter problems of rectangles, triangles, and circles
- Select enemies by clicking on their sprite or clicking on their problem
- Defeat waves of enemies and bosses which consist of many problems
- Earn currency for defeating enemies and bosses, and accumulate score for doing so.
- Purchase upgrades in the shop which give them a permanent advantage for the rest of the run.
- Purchase powerups in the shop which they can use to get a temporary advantage.
- Strategize and defeat multiple enemies with one problem with the Explosion mechanic.
- View helpful formulas and change settings involving audio and tutorials
- Store data between closing the application and loading it to keep track of settings, player name, and unlocks

# How to Run (Windows Only)
1. Visit <a href="https://aerialareas.github.io/Aerial-Areas/">our website</a> and click the download link titled 'Download the Game(Prototype V2)' at the bottom of the page.

2. Extract the zip file and run AerialAreas.exe.

Note: The game is currenlty only built for one resolution

# How to Build (Windows Only)
1. Clone this repository to your local machine

2. Install Godot_v4.5.1-stable_mono_win64

3. Select 'Import' in Godot and navigate to the 'game' folder from this repository

4. Click 'Build' or 'Run' in the Godot Editor
